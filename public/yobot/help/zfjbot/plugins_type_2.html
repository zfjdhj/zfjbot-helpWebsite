# 公会排名查询

指令功能
会战排名+公会名/排名查看指定名字或指定排名的公会排名
会战锁定+公会名绑定本群的公会
会战解锁+公会名解绑本群的公会
公会排名查询本群绑定的公会的排名信息
(自动)公会排名推送每天早5点30分自动推送本群绑定的公会的排名信息

# 公会装备捐助提醒

指令功能

(自动)装备请求提醒每当有新请求，或者临近结束仍未完成时触发

invite check查看白名单玩家公会信息
